Challenge 1 - Syllabes

Next »

As an elementary school teacher, you are explaining some basic concepts to your class. They are just starting, so you decided to simplify things a bit:

  • We define a syllabe as a string of characters of any length, of which only the first or the last one (and never both at the same time) can be a consonant and at least one has to be a vowel.
  • A consonant is any letter different than the following: a e i o u y
  • A word is the concatenation of one or more syllabes.

After this, you gave them an assignment to practise: writing a list of words. As expected though, they aren't doing so well - it's their first time after all and most of the words are not valid!

You need to point out their mistakes, but kids are very sensitive these days, so you should try to minimize the amount of corrections that you apply to each word.

For example, you can correct the string wrong with two corrections: changing the letters r and n to vowels. Notice that a correction consists in changing a single letter for a different one and never in adding or substracting letters.

Input

The first line will contain an integer C, the number of cases for our problem.
Each case consists of a line with a single string, formed by lowercase english letters.

Output

For each case, a line starting with "Case #x: " followed by the minimum number of corrections. Every line is followed by a new line character.

Examples

Case 1:

yea

Case 2:

isalsook

Case 3:

bcdf

Case 4:

bcdfg

Case 5:

abcdefghijklypq

In the Case 1 the string is already a valid word.
In the Case 2 the string is already a valid word.
In the Case 3 all the letters are consonants, we need to change two of them.
In the Case 4 we need to change three of the letters, similar to Case 3.
In the Case 5 the minimum amount of corrections is 4.

Limits for the test phase

  • 1 ≤ Word length ≤ 50

Limits for the submit phase

  • 1 ≤ Word length ≤ 10000

Sample Input

5
yea
isalsook
bcdf
bcdfg
abcdefghijklypq

Sample Output

Case #1: 0
Case #2: 0
Case #3: 2
Case #4: 3
Case #5: 4
